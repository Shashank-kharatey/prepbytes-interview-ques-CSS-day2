Q.1 what is box sizing?

Ans.1 'Box sizing' is the property in css which allows us to set the instruction for calculating the actual width/height of the box/element.

By default the actual width/height of the box/element is the sum of padding given,the border given and the width/height given. Using the box-sizing 'border-box' property the width given do not changes and the padding and border given will be adjusted coresponding to the actual width of the content within the width given to the box/element.

Q.2 what is animation delay?

Ans.2 'animation-delay' is the property that defines the time to be taken by the animation to start after the webpage is loaded on the browser.

Q.3 which property you will use to merge the cells vertically in grid?

Ans.3 In order to merge the cells vertically in the grid, we will use the property grid-row-start:('starting value from which the spaning should start'); grid-row-end:('ending value upto which the spaning should end'); shorthand-(grid-row:'start value'/'end value');

Q.4 which property you will use to merge the cells horizontally in grid?

Ans.4 In order to merge the cells horizontally in the grid, we will use the property grid-column-start:('starting value from which the spaning should start'); grid-column-end:('ending value upto which the spaning should end'); shorthand-(grid-column:'start value'/'end value');

Q.5 Explain CSS box model?

Ans.5 All the element/container/identity present on the webpage folows the pricipal of box-mode, according to which the outer most part/space of the element is known as margin whose value does included in the actual size of the element rather it only covers the element, after the margin there it comes the border around the element whose value is included in the actual size of the element(by default), after that comes the padding around and its value is also included in the actual size of the element, at last there comes the content of the element at the center.

Q.6 what is the differnce between display none and visibility hidden?

Ans.6 The basic difference between the two is:- If we want to remove the box/element completely(its content and space both) we should use display:none;
But, if we want to remove the content but allow the space occupied by the box/element to be present in on page, we should use visibility:hidden;
